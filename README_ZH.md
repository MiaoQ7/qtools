# QTOOLS

## 🏆 Start

- Choose either **yarn** or **npm**

**🍬 Clone**

```bash
https://github.com/MiaoQ7/qtools.git
```

```bash
cd qtools
```

**🍙 Install**

```bash
yarn
```

```bash
npm install
```

**🌽 Run**

```bash
yarn go
```

```bash
npm run go
```

**🍭 Build**

```bash
yarn buildapp
```

```bash
npm run buildapp
```
